Formulario Flask-WTForms
- é uma extensão do Flask que torna o processamento de formularios mais fácil de gerenciar.
- valida a entrada do usário
- fornece formulário seguro com token CSRF
- renderiza os elementos do formulario dinamicamente.
- fornece upload de arquivo que funciona com o Flask Uploads.
- tanto o Flask, quanto o Django, fazem uso dos formulários WTForms.


CSRF = Cross-Site Request Forgery - é um tipo de ataque de websites maliciosos.
- também chamado de ataque de um clique ou transporte de sessão.
- ocorre quando uma requisição HTTP é feita entre sites na tentativa de se passar por um usuário legitimo
